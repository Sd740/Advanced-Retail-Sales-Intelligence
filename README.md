# Advanced Retail Sales Intelligence Project
## Tools Used
- Excel
- SQL
- Python
- Power BI

## Project Overview
This project analyzes retail sales data to generate KPIs, identify top customers, and visualize business performance using Power BI.

## Key Metrics
- Total Revenue
- Total Orders
- Average Order Value
- Monthly Sales Trend

## Features
- Data Cleaning using Python
- Business Queries using SQL
- Interactive Dashboard in Power BI
